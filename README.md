# Homebrew Tap for SilbercueSwift

The fastest MCP server for iOS development.

## Install

```bash
brew tap silbercue/silbercue
brew install silbercueswift
```

## What's included

**SilbercueSwift** — 40 MCP tools for iOS development in one 8.5MB Swift binary:

- Build & Test with xcresult parsing (structured results, not raw logs)
- UI Automation via direct WebDriverAgent (no Appium needed)
- Screenshots in 0.3s (44x faster than XcodeBuildMCP)
- Code coverage, failure screenshots, log pattern matching

More info: [github.com/silbercue/SilbercueSwift](https://github.com/silbercue/SilbercueSwift)
